# AI Employee Console

Run:

```bash
python app.py
```

Open `http://localhost:8080`.

## What this demonstrates

A cross-functional onboarding AI employee that perceives a messy trigger, reasons across HR/IT/Finance, acts against mocked systems, escalates ambiguity, and learns company-specific decisions.

## Hero moment

Priya Sharma teaches the system that `Sales Engineer = Sales`. Omar Reyes later benefits from that learned rule automatically: no escalation, a Company Brain match, and a visible rule-fire animation.

## Company Brain thesis

The system becomes more useful one human decision at a time. The correction store is persistent, queryable, and grounded only in decisions the company has actually made.
